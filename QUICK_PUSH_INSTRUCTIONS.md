## Troubleshooting

### Error: "Permission denied (publickey)"
**SSH key not set up.** Use HTTPS method instead:
```bash
git remote add origin https://github.com/sdas/healthexpert.git
git push -u origin main
```

### Error: "fatal: remote origin already exists"
```bash
git remote remove origin
git remote add origin git@github.com:sdas/healthexpert.git
git push -u origin main
```

### Error: "The authenticity of host 'github.com' can't be established"
Answer **yes** when asked to continue connecting.
