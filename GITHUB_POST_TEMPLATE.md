
Models:

Storage:

```
Hi ML community! 👋

I'm excited to share HealthExpert, an open-source platform I've been building 
for intelligent document analysis.

## What is HealthExpert?

An enterprise-grade AI system that combines:
- Multi-agent orchestration (CrewAI)
- Hybrid retrieval (ChromaDB + Neo4j)
- Multi-format document support (PDF, DOCX, XLSX, CSV, TXT, Images)
- KV-cache optimization for full-document reasoning
- Production-ready web UI and CLI

## Why I Built It

Current document analysis solutions are either:
1. Too simple (just keyword search)
2. Too slow (full re-computation per query)
3. Missing entity relationships (no graph DB)
4. Not production-ready

HealthExpert solves all of these.

## Tech Stack

- **LLM**: Qwen3-8B
- **Embeddings**: BAAI/bge-m3
- **Agents**: CrewAI 0.36+
- **Vector DB**: ChromaDB
- **Graph DB**: Neo4j
- **Web**: Flask + SSE
- **Infrastructure**: Docker

## Performance

- Document ingestion: 2.5s per 10-page PDF
- Query with KV-cache: 0.8s
- Full pipeline: 6.5s
- Embedding throughput: 312 chunks/sec
```
